Chapter 4 Drawing Sheet Layout

[cite_start]Standard layouts are specified by various organizations and typically include a drawing frame, title block, parts list (bill of materials), and revision table[cite: 198, 199].

4.0.1 Sheet Frames

[cite_start]A drawing frame defines a margin around the outside of the drawing area[cite: 231, 232]. [cite_start]Margins are standardized for each paper size[cite: 233].

4.0.1.1 ISO Standard Paper Sizes (ISO 216)

[cite_start]The following table outlines standard ISO sizes and their drawing frame dimensions[cite: 241, 255]:

Paper Size Border Width (mm) (Left/Top/Right/Bottom) Frame Dimensions (mm) (Width x Height)
A0 28 / 20 1133 x 801
A1 20 / 14 801 x 566
A2 14 / 10 566 x 400
A3 7 / 10 4003 x 283
A4 7 / 5 283 x 200

4.0.2 Title Block

[cite_start]Located in the bottom right, it should contain[cite: 257]: * [cite_start]Company name and drawing title[cite: 258, 259]. * [cite_start]Unique drawing number and scale[cite: 260, 261]. * [cite_start]Drawing size and angle of projection symbol[cite: 262, 263]. * [cite_start]Signatures/initials of draftsman and checkers with dates[cite: 264, 265]. * [cite_start]Material, revision number, and sheet number[cite: 266, 269, 270].

[cite_start]For CAD drawings, it is recommended to include the software name, file name, and dimension units[cite: 272, 274, 275, 277].

4.0.3 Material or Parts List (Bill of Materials)

[cite_start]Used for assembly drawings to list all parts, usually placed just above the title block[cite: 280, 281]. It includes: * [cite_start]Part number and name[cite: 288, 289]. * [cite_start]Quantity and material specifications[cite: 290, 291]. * [cite_start]The drawing number for each individual part[cite: 292].

4.1 Zoning and Notes

  • [cite_start]Zoning: A grid system (letters and numbers) inside the frame for easy reference on large sheets[cite: 295, 296, 297].
  • [cite_start]Notes: Information necessary for the drawing other than pictorial views and dimensions, usually placed in the lower-left corner[cite: 298, 299].
